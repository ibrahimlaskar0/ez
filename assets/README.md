# Assets Directory

This directory contains media assets for the Esplendidez 2026 website.

## Current Assets

### Videos
- `video-bg.mp4` - Background video for hero sections

### Images (images/)
- Placeholder images for gallery
- Sample event photos
- QR code for payments

### Required Assets

To fully complete the website, you may want to add:

1. **Payment QR Code** (`qr-demo.png`)
   - UPI QR code for payment processing
   - Recommended size: 300x300px

2. **Gallery Images**
   - Event photos from previous editions
   - Competition highlights
   - Cultural event moments

3. **Event Icons**
   - Custom icons for different event categories
   - Logo variations

4. **Video Thumbnail** (`video-thumbnail.jpg`)
   - Poster image for video element
   - Should represent the video content

## Note
The website will function without these assets as it includes fallback designs and placeholder elements.
